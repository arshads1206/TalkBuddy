# Why each modification was necessary
- **Platform Abstraction (`src/renderer/platform/`)**: Created to seamlessly swap between `window.electronAPI` (Sandbox mode) and `fetch()` (Browser mode) without touching React UI code.
- **FastAPI Bridge (`main.py`)**: Created to proxy requests securely around Browser CORS and handle network lifecycle management on Windows.
- **Node Helper (`run_db_op.js`)**: Allowed us to re-use identical SQLite DB logic without needing to reinvent the DB models/schemas in Python.
- **TTS Subprocess Fix (`server.py`)**: Required forcing standard I/O encoding to prevent Piper crashes when processing complex unicode punctuation (like em-dashes) from the LLM.
- **Vite Config (`vite.config.mts`)**: Required a local dev proxy to map `/api` transparently to FastAPI.
